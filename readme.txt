== Description ==

[WPLoyalty - Auto Currency](https://wployalty.net?utm_campaign=wpl_auto_currency&utm_source=gitrepo&utm_medium=listing) used convert amount automatically.

== Installation ==

Just use the WordPress installer or upload to the /wp-content/plugins folder. Then Activate the WPLoyalty - Auto currency plugin.
More information could be found in the documentation

= Minimum Requirements =

    - WordPress 4.9.0 or greater
    - WooCommerce 6.5.0 or greater
    - WPLoyalty 1.2.1 or greater
    - PHP version 7.0.0 or greater
